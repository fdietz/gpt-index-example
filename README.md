# GPT Index example

Example python index and query using [GPT Index](https://gpt-index.readthedocs.io/en/latest/).

Based on the following [blog article](https://dev.to/dhanushreddy29/fine-tune-gpt-3-on-custom-dataset-with-just-10-lines-of-code-using-gpt-index-18mc).

Example PDF downloaded from [Can Artificial Intelligence Transform DevOps?](https://arxiv.org/abs/2206.00225v1).

You need an `OPENAI_API_KEY`.

## Index

```bash
OPENAI_API_KEY=<your-api-key> python3 main.py
```

## Query

```bash
OPENAI_API_KEY=<your-api-key> python3 query.py
```
